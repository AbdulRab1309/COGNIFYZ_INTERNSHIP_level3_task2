# 🚀 Cognifyz Web Developer Internship Landing Page

A modern, responsive, and visually appealing internship landing page built using **HTML5** and **CSS3**. This project showcases the **Cognifyz Web Developer Internship Program** through a clean, professional, and minimalist user interface inspired by modern corporate web design principles.

---

## 📌 Project Overview

This project is a single-page internship landing page designed to provide information about the Cognifyz Web Developer Internship Program. It presents details about the company, internship program, qualifications, benefits, and application information in a structured and engaging format.

The design focuses on:

* Minimalist aesthetics
* Responsive layouts
* Modern typography
* Clean user experience
* Professional presentation of internship opportunities

---

## ✨ Features

### 🎯 Responsive Design

* Fully responsive layout for desktop, tablet, and mobile devices.
* Adaptive grid system for improved user experience across screen sizes.

### 🎨 Modern UI/UX

* Clean black-and-white theme.
* Professional typography using Helvetica-based fonts.
* Smooth hover effects and interactive elements.

### 📑 Structured Content Sections

* About Us
* Program Overview
* Internship Highlights
* Qualifications
* Call-to-Action Section
* Footer with Contact Information

### 📱 Mobile-Friendly Layout

* Responsive grid transformation for smaller screens.
* Optimized spacing and readability.

### 🔝 Sticky Navigation Bar

* Fixed navigation at the top.
* Easy branding visibility throughout scrolling.
